# Правила оформления коммитов

Сообщение коммита:
- На **русском языке**
- Одно предложение
- Глагол в **прошедшем времени от первого лица**
- Описывает выполненное действие

**Примеры:**
- «Добавил браузерный клиент для Web Push»
- «Исправил ошибку валидации формы»
- «Переработал архитектуру планировщика»
- «Обновил зависимости до актуальных версий»

## Процесс

1. Выполни `git status` и `git diff` чтобы понять что изменилось
2. Сформулируй сообщение по правилам выше
3. Сделай коммит с указанием себя как соавтора:

```
Co-Authored-By: {название используемой модели} <noreply@anthropic.com>
```
