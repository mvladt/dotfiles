# Глобальные правила

## Markdown-таблицы

При создании или редактировании таблиц в `.md`-файлах — выравнивай столбцы пробелами, чтобы пайпы были в столбец и таблица читалась в raw-виде.

## Frontend

- **Нативный CSS вместо препроцессоров** — использовать обычный CSS (`<style scoped>`), SCSS/SASS только если нативный CSS объективно не справляется с задачей
- **E2E тесты** — smoke-тест на уровне браузера: проверяем, что ключевые флоу работают от начала до конца. Детали — на уровне unit/integration
